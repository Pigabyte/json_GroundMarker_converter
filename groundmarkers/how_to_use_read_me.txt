1) Make sure you have a 3.2+ Version of Python (Latest if Possible)
https://www.python.org/downloads/
2) Press: Windows Key + R 
2) Type: .runelite (Press Enter)
3) Select your main profile
4) Copy your file of 'settings.properties' to the folder 'groundmarkers'
5) Run the File 'Convert.bat'
6) Open HDOS Client and Login
7) Press: Shift Twice Quickly
8) Type: Ground Markers
9) Activate the Plugin
10) If you want -> Reset Plugin (This will Clean your Current Markers)
11) Open 'FileGenerated.txt' in folder 'groundMarkers'
12) Right Click -> Select All -> Copy (This Will Copy to Clipboard)
13) Go back to HDOS Client
14) Press 'Import From Clipboard)
15) If it says 'Imported (number) tile markers!' you have do it.
